Algerian Forest Fires Prediction

This project uses the Algerian Forest Fires dataset to predict the Fire Weather Index (FWI) based on various weather and environmental factors.

Dataset Information

The dataset includes 244 instances that regroup data from two regions of Algeria, namely the Bejaia region located in the northeast of Algeria and the Sidi Bel-abbes region located in the northwest of Algeria.

The period from June 2012 to September 2012. The dataset includes 11 attributes and 1 output attribute (class). The 244 instances have been classified into fire (138 classes) and not fire (106 classes) classes.

Attribute Information

  • Date: (DD/MM/YYYY) Day, month ('June' to 'September'), year (2012)
  • Temperature: temperature noon (temperature max) in Celsius degrees: 22 to 42
  • Relative Humidity (RH): in %: 21 to 90
  • Wind Speed (Ws): in km/h: 6 to 29
  • Rain: total day in mm: 0 to 16.8
  • Fine Fuel Moisture Code (FFMC) index from the FWI system: 28.6 to 92.5
  • Duff Moisture Code (DMC) index from the FWI system: 1.1 to 65.9
  • Drought Code (DC) index from the FWI system: 7 to 220.4
  • Initial Spread Index (ISI) index from the FWI system: 0 to 18.5
  • Buildup Index (BUI) index from the FWI system: 1.1 to 68
  • Fire Weather Index (FWI) Index: 0 to 31.1
  • Classes: two classes, namely Fire and not Fire
